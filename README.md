# Benoir Client Portal

Internal client portal for Benoir Travel — manage bookings, hotel search, pricing, finance, and support.

## Running locally

Requires Python 3 (pre-installed on macOS/Linux; download from python.org on Windows).

```bash
# Clone
git clone https://github.com/ilyaoguy/benoir-client-portal.git
cd benoir-client-portal

# Start server
python -m http.server 3456

# Open in browser
http://localhost:3456
```

No build step, no dependencies.
